## 第一部分:前端美化(升级版)

上一轮已定了深色科技风的配色 token,本轮在那基础上**做精**,重点是高清、对比清晰、有质感。

### 1. 视觉系统(在已有 token 上强化)
- 背景:深炭黑 `#0d1117`,卡片 `#161b22`,可叠加极淡的网格或径向光晕背景图(素材见下)提升科技感,但**透明度要低(≤8%)**,绝不能影响文字可读。
- 文字:正文 `#e6edf3`(对比度足够),次要 `#9da7b3`,标题可用纯白 `#ffffff` 加重量感。
- 强调色:科技蓝 `#2f81f7` + 青色点缀 `#39d0d8`,用于按钮、关键数据、图表线条。
- **对比度硬指标**:所有正文/背景对比度 ≥ 4.5:1。CC 实施后逐页自查,凡是看不清的一律加深背景或提亮文字。

### 2. 排版与组件
- 用卡片化布局把"基本面/技术面/情绪面/风控"分区,每块卡片有标题条 + 图标。
- 关键数字(MA5、收盘价、置信度)用大号 + 强调色突出。
- 图表(K线、情绪曲线)用统一配色,深色底 + 亮色线,坐标轴文字要够亮。
- 加载/分析中:用骨架屏或进度条 + "分析中,请勿刷新页面"提示(呼应上一轮的刷新中断问题)。
- 字体:`"Segoe UI", "Microsoft YaHei", system-ui` 保证中文清晰;数字可用等宽字体增加专业感。

### 3. 高清化
- 所有图标用 **SVG**(矢量,任意缩放不糊),不要用低分辨率 PNG。
- 如用位图背景,提供 @2x 分辨率,适配高 DPI 屏。

## 第三部分:K线分析模块的"无显卡/无N卡"适配

你的核心疑问:大部分用户没独立显卡(更没 N 卡),是不是就用不了 K线深度学习预测(Kronos)?

### 结论:可以用,而且不难。Kronos 完全能跑 CPU。
- Kronos 的 small/base 模型很小(参数量远小于 LLM),**CPU 推理是可行的**,只是比 GPU 慢——对"预测未来几日K线"这种**非实时、单次**的任务,慢几秒到几十秒完全可接受。
- 关键认知纠正:**"CPU 版 CUDA"这个说法不成立**。CUDA 是 N 卡专属,没有"CPU 版 CUDA"。正确做法是——**装 CPU 版的 PyTorch**(`pip install torch` 的 CPU wheel),模型用 `device="cpu"` 跑。N 卡用户才装 CUDA 版 PyTorch。

### 实施方案:自动检测设备 + 三档适配
在 Kronos 微服务启动时自动选 device:
```python
import torch

def pick_device():
    if torch.cuda.is_available():
        return "cuda"          # N卡(你的5070Ti)
    # 苹果M系列
    if getattr(torch.backends, "mps", None) and torch.backends.mps.is_available():
        return "mps"
    return "cpu"               # 无独显/AMD/Intel核显 → CPU,人人可用
```
- **N 卡用户**:走 cuda,最快。
- **Mac(M芯片)用户**:走 mps。
- **无独显 / AMD / Intel 用户**:走 cpu,照样能出预测,只是慢些。

### 打包与依赖策略(重要)
PyTorch 的 CUDA 版体积巨大(含 CUDA 库,GB 级),而绝大多数用户用不上。建议:
1. **默认打包 CPU 版 PyTorch**(体积小、人人能跑),保证"开箱即用 K线分析"。
2. **N 卡用户想要加速**:做成"可选 GPU 加速包"——配置里给个开关/单独下载,引导他们装 CUDA 版 torch。或在文档里说明如何升级。
3. Kronos 模型权重(small/base)体积不大,可随包或首次使用时下载。

### 体验设计
- K线预测模块**默认开启**(因为现在 CPU 也能跑了),不再依赖 GPU。
- UI 显示当前算力模式:"GPU 加速 / CPU 模式(较慢)",并在 CPU 模式下给进度提示。
- 预测耗时长时用后台任务 + 进度条,避免卡住界面。
- 始终保留"关闭K线预测"的降级开关(老机器/赶时间时)。

### 给 CC 的落地要点
- requirements / spec 默认用 CPU 版 torch。
- Kronos 服务用上面的 `pick_device()`,严禁硬编码 `cuda`。
- 文档说明:N卡用户如何切换到 GPU 加速版获得更快速度。
