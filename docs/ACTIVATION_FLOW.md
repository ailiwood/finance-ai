# QuantSage 安装 → 购买 → 激活 → 使用 完整流程

> 面向普通用户的端到端操作指南。开发者也可用此文档向用户说明完整流程。

---

## 一、安装软件

### 1.1 获取安装包

开发者会通过以下方式之一提供安装包：

- 飞书群文件分享
- 网盘链接（百度网盘 / 阿里云盘）
- 直接发送 `QuantSage_Setup_v1.0.0.exe`

### 1.2 运行安装程序

1. 双击 `QuantSage_Setup_v1.0.0.exe`
2. 如果 Windows 弹出 "Windows 已保护你的电脑" 提示，点击 **"更多信息"** → **"仍要运行"**
3. 安装向导共 4 步：

| 步骤 | 页面 | 操作 |
|------|------|------|
| 第 1 步 | 欢迎页 | 阅读功能介绍，点击"下一步" |
| 第 2 步 | 安装选项 | 勾选需要的组件（桌面快捷方式、开始菜单），点击"下一步" |
| 第 3 步 | 许可协议 | **必须**完整阅读协议，勾选两个同意选项后才能继续 |
| 第 4 步 | 安装 | 点击"安装"，等待完成 |

4. 安装完成后，可选择"启动 QuantSage"

> 默认安装路径：`%LOCALAPPDATA%\Programs\QuantSage\`
> 快捷方式：桌面 + 开始菜单（如已勾选）

## 二、首次启动与免责确认

### 2.1 启动软件

- 双击桌面快捷方式，或从开始菜单启动
- 软件启动后会弹出命令行窗口（黑色背景，显示启动日志）
- **不要关闭命令行窗口**，它是服务器进程
- 等待浏览器自动打开（约 5-10 秒），如果没有自动打开，查看命令行窗口显示的网址（如 `http://127.0.0.1:8501`），手动在浏览器中输入

### 2.2 免责声明确认

首次启动时，页面会显示免责声明弹窗：

> ⚠️ 本软件仅供参考研究，不构成任何投资建议，盈亏自负。

- **必须点击"我已知晓并同意"** 才能继续
- 此确认只需一次，后续启动不再显示

## 三、配置 API Key

### 3.1 为什么需要 API Key

QuantSage 的分析能力依赖大语言模型（DeepSeek）。你需要一个 DeepSeek API Key 才能使用分析功能。

### 3.2 获取 DeepSeek API Key

1. 打开 https://platform.deepseek.com/
2. 注册账号（手机号即可）
3. 进入"API Keys"页面
4. 点击"创建 API Key"
5. 复制生成的 Key（格式：`sk-xxxxxxxxxxxxxxxx`）

> DeepSeek API 按使用量付费，费用极低。一次完整股票分析约消耗 0.1-0.3 元人民币。

### 3.3 配置向导

1. 同意免责声明后，自动进入 3 步配置向导
2. **第 1 步 — API Key**：粘贴你的 DeepSeek API Key
3. **第 2 步 — 数据源**：选择数据源（默认 AkShare 免费即可）
4. **第 3 步 — 偏好设置**：选择风险偏好（保守/平衡/积极）和分析深度
5. 点击"完成配置"

## 四、激活许可证

> QuantSage 需要有效许可证才能使用全部功能。未激活状态下部分功能受限。

### 4.1 付费购买

1. 在软件激活页面查看你的 **设备码**（16 位十六进制字符，如 `70E3032EA0E74D51`）
2. 复制设备码
3. 使用支付宝扫描激活页面上显示的收款码
4. **付款时务必在备注中填写你的设备码**
5. 付款金额：¥19.90（或开发者设定的价格）

### 4.2 提交订单

1. 点击软件中的"打开激活网页"按钮（或手动访问 https://quantsage-activation.lk166564317.workers.dev/）
2. 在"付款激活"标签页，输入你的设备码
3. 点击"提交订单"
4. 页面显示"订单已创建，请等待激活码"

### 4.3 获取激活码

1. 开发者收到支付宝到账通知后，核对设备码
2. 开发者在管理后台签发激活码（通常 5 分钟内完成）
3. 你回到激活网页，切换到"查询激活码"标签页
4. 输入设备码，点击"查询"
5. 如果激活码已就绪，页面会显示一长串以 `QS` 开头的密钥
6. 点击"复制激活码"

### 4.4 完成激活

1. 回到 QuantSage 软件激活页面
2. 将激活码粘贴到"许可证密钥"输入框
3. 点击"激活"
4. 显示激活成功后，进入主界面

> ⚠️ 激活码绑定设备，一码一机。换电脑或重装系统后设备码可能变化，需联系开发者重新激活。

## 五、使用软件进行分析

### 5.1 分析单只股票

1. 进入主界面后，在"股票分析"区域：
   - 选择市场（A股/美股/港股）
   - 输入股票代码（如 `600519`）
   - 选择分析深度（1-5，推荐 3）
2. 点击"开始分析"
3. 等待 3-5 分钟（分析过程中会显示进度）
4. 分析完成后页面展示完整研究报告

### 5.2 阅读报告

报告包含以下章节（按顺序）：

| 章节 | 内容 |
|------|------|
| 技术面分析 | K线走势、均线、MACD、RSI、KDJ、BOLL |
| 基本面分析 | 公司信息、行业分类、PE/PB 估值框架 |
| 投资者情绪分析 | 基于真实新闻和公告的情绪判断 |
| 风险管控与最终研究结论 | 多智能体风险评估和决策辩论 |
| Kronos 深度学习 K 线预测 | 概率性价格预测（仅供研究参考） |
| 综合结论 | AI 多智能体综合观点 + Kronos 交叉验证 |
| 多周期技术指标汇总 | MA/MACD/RSI/KDJ/BOLL 多周期数值 |
| 免责声明 | 风险提示（始终在报告末尾） |

### 5.3 下载报告

- **Markdown 下载**：纯文本格式，可在任意文本编辑器打开
- **PDF 下载**：排版精美的 PDF 文档，适合打印和分享

### 5.4 历史报告

- 每次分析完成后自动保存到 `~/.quantsage/reports/`
- 点击主页"历史报告"按钮查看所有历史分析

## 六、常见问题

### Q: 软件打不开 / 没有反应
A: 检查命令行窗口是否有错误信息。确认已安装最新版，或联系开发者。

### Q: 分析到一半卡住了
A: DeepSeek API 可能暂时繁忙，等待即可。如果超过 10 分钟无响应，重启软件重试。

### Q: 分析结果显示不完整
A: 尝试将分析深度调低至 2-3，或检查网络连接。

### Q: 换电脑了怎么办
A: 设备码会变化，需联系开发者重新签发激活码。

### Q: API Key 在哪里获取
A: https://platform.deepseek.com/ → 注册 → API Keys → 创建
