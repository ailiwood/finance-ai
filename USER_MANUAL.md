## 一、软件简介

### 1.1 什么是 QuantSage

QuantSage 是一款**本地运行的 Windows 桌面股票研究辅助软件**。它使用多智能体 AI 架构，自动采集公开市场数据，生成包含技术面、基本面、情绪面和深度学习预测的综合研究报告。

### 1.2 核心功能

| 功能 | 说明 |
|------|------|
| **多智能体 AI 分析** | 8 个专业 AI Agent 协作，模拟市场分析师、基本面分析师、情绪分析师、多空辩论、风险管理等角色 |
| **技术面分析** | 真实 K 线数据（前复权），MA/MACD/RSI/KDJ/BOLL 多周期技术指标 |
| **基本面分析** | 公司基本信息、行业分类、估值框架（PE/PB/ROE） |
| **情绪面分析** | 基于真实新闻公告的情绪判断，非 AI 编造 |
| **K 线预测** | Kronos-base 深度学习模型（102.3M 参数，MIT 开源），概率性价格预测 |
| **多源数据** | BaoStock → AKShare → Tushare → 东方财富，自动降级切换 |
| **报告导出** | Markdown 文本下载 + PDF 精美排版导出 |
| **历史存档** | 每次分析自动保存，可随时查阅过往报告 |

### 1.3 不支持的功能（明确声明）

- ❌ 不提供实盘下单或连接券商
- ❌ 不提供自动交易或程序化交易执行
- ❌ 不提供"稳赚"、"必涨"等保证性结论
- ❌ 不提供投资建议或荐股服务
- ❌ 不提供实时行情推送

开发者会通过飞书群、网盘或其他方式提供安装包文件：
- 文件名：`QuantSage_Setup_v1.0.0.exe`

### 3.2 安装步骤

1. 双击 `QuantSage_Setup_v1.0.0.exe` 启动安装向导
2. 如果 Windows SmartScreen 弹出提示，点击 **"更多信息"** → **"仍要运行"**
3. **第 1 步 — 欢迎页**：阅读功能介绍，点击"下一步"
4. **第 2 步 — 安装选项**：
   - 核心程序（必需，不可取消）
   - 创建桌面快捷方式（推荐勾选）
   - 添加到开始菜单（推荐勾选）
   - 安装完成后启动 QuantSage（可选）
5. **第 3 步 — 许可协议**：**必须完整阅读**软件许可协议与风险告知，勾选以下两项：
   - ☑ 我已完整阅读并同意上述软件许可协议
   - ☑ 我已知晓并接受全部投资风险，承诺盈亏自负
6. **第 4 步 — 安装**：点击"安装"，等待进度条完成（约 1-2 分钟）

### 3.3 安装位置

```
默认安装路径：%LOCALAPPDATA%\Programs\QuantSage\
用户数据目录：%USERPROFILE%\.quantsage\
  ├── .env              API Key 等配置
  ├── license.json      许可证信息
  ├── logs/             运行日志
  ├── reports/          历史报告存档
  └── cache/            数据缓存
```

### 3.4 卸载

1. 打开 Windows "设置" → "应用" → "已安装的应用"
2. 搜索 "QuantSage"
3. 点击"卸载"
4. 或运行安装目录下的 `unins000.exe`

> 卸载不会删除 `%USERPROFILE%\.quantsage\` 下的个人数据。如需彻底清除，请手动删除该目录。

## 四、首次启动与配置

### 4.1 启动软件

- 双击桌面上的 **QuantSage** 快捷方式
- 软件启动后会出现一个**命令行窗口**（黑色背景，显示服务器日志）
- **请勿关闭命令行窗口**，关闭它将停止软件运行
- 浏览器会自动打开分析页面（如未自动打开，见下方说明）

### 4.2 浏览器未自动打开时

1. 查看命令行窗口中的 URL 地址（如 `http://127.0.0.1:8501`）
2. 手动在浏览器地址栏输入该地址
3. 推荐浏览器：Chrome、Edge、Firefox

### 4.3 免责声明确认

首次启动时，页面显示免责声明：

> ⚠️ 本软件基于 AI 模型生成内容仅供参考研究，不构成任何投资建议。  
> 所有分析观点、价格预测均为概率性结果，不保证准确性。  
> 投资者应独立判断，盈亏自负。

**必须点击"我已知晓并同意"才能继续。**

### 4.4 配置 API Key

QuantSage 需要 DeepSeek API Key 才能驱动 AI 分析：

**获取 API Key：**
1. 访问 https://platform.deepseek.com/
2. 注册账号（支持手机号）
3. 进入左侧 "API Keys" 页面
4. 点击 "创建 API Key"，命名后复制

**配置：**
1. 在软件 3 步配置向导的第 1 步粘贴 API Key
2. 第 2 步选择数据源（默认 AkShare 即可）
3. 第 3 步选择风险偏好和分析深度

> DeepSeek API 按量计费，一次完整分析约 0.1-0.3 元，新用户通常有免费额度。

### 4.5 测试配置

配置完成后，在主页输入股票代码（如 `600519` 贵州茅台），选择分析深度 1（快速测试），点击"开始分析"。如果 2-3 分钟内看到分析报告，说明配置成功。

## 六、股票分析

### 6.1 开始分析

1. 在主页"股票分析"区域：
   - **选择市场**：A 股（上海/深圳）、美股（NASDAQ/NYSE）、港股（HKEX）
   - **输入代码**：如 `600519`（贵州茅台）、`AAPL`（Apple）、`00700`（腾讯）
   - **分析深度**：1（最快/最简）~ 5（最详细/最慢），推荐 3
2. 点击 **"开始分析"**
3. 等待分析完成（3-5 分钟）
4. 页面自动展示完整研究报告

### 6.2 分析深度说明

| 深度 | 辩论轮次 | 风险讨论轮次 | 预计耗时 | 适用场景 |
|------|---------|-------------|---------|---------|
| 1 | 1 | 1 | ~2 分钟 | 快速浏览 |
| 2 | 1 | 1 | ~3 分钟 | 简要分析 |
| 3 | 1 | 1 | ~3-5 分钟 | 日常使用（推荐） |
| 4 | 2 | 2 | ~6-8 分钟 | 深入研究 |
| 5 | 2 | 3 | ~10-12 分钟 | 全面分析 |

### 6.3 A 股交易日说明

A 股交易日为**周一至周五（法定节假日除外）**，交易时间 9:30-15:00。

- 在非交易日进行分析时，获取的 K 线数据为最近一个交易日的数据
- 节假日无数据属于正常现象，不是软件故障

## 七、阅读与导出报告

### 7.1 报告结构

每份研究报告按以下固定顺序排列：

| 章节 | 内容说明 |
|------|---------|
| **技术面分析** | K 线走势、MA/MACD/RSI/KDJ/BOLL 信号、量价关系 |
| **基本面分析** | 公司信息、行业分类、PE/PB 估值参考 |
| **投资者情绪分析** | 新闻公告情绪判断、市场关注度 |
| **风险管控与最终研究结论** | 多空辩论结果、风险管理决策 |
| **Kronos 深度学习 K 线预测** | 概率性价格预测（或模型状态说明） |
| **综合结论** | AI 多智能体综合观点，含 Kronos 交叉验证 |
| **多周期技术指标汇总** | MA/MACD/RSI/KDJ/BOLL 数值表 |
| **免责声明** | 风险提示（始终为报告最后一段） |

### 7.2 Markdown 下载

- 点击报告下方的 "下载 Markdown" 按钮
- 生成 `.md` 格式纯文本文件
- 可在记事本、VS Code、Typora 等任意文本编辑器打开
- 文件名格式：`{股票代码}_报告.md`

### 7.3 PDF 下载

- 点击报告下方的 "下载 PDF" 按钮
- 生成排版精美的 PDF 文档
- 包含中文宋体/黑体字体、表格、分隔线
- 适合打印存档或分享
- 文件名格式：`{股票代码}_报告.pdf`

### 7.4 历史报告

- 所有分析自动存档到 `~/.quantsage/reports/`
- 点击主页"历史报告"按钮查看历史列表
- 每条记录显示：分析时间、股票代码、分析结论

## 八、高级功能

### 8.1 数据体检

主页点击"数据体检"按钮，可以对指定股票的 K 线数据进行校验：

- 与同花顺/东方财富的数据对账
- 验证前复权价格准确性
- 检查 MA5 等常用指标计算是否正确

### 8.2 合规扫描

主页点击"合规扫描"按钮，可以对当前报告文本进行合规检查：

- 检测是否含有"买入"、"卖出"等违规措辞
- 验证是否包含免责声明
- 确保报告符合合规要求

### 8.3 日志监控

主页展开"实时日志监控"区域，可以查看：

- 软件运行日志
- 分析过程中的实时进度
- 错误和警告信息

### 8.4 GPU 升级（可选）

如果你有 NVIDIA 显卡（GTX 10 系列及以上），可以在"系统状态"面板点击"升级 GPU 版"：

- 自动安装 CUDA 版 PyTorch（下载约 2.5 GB）
- 升级后 Kronos K 线预测速度提升约 3-5 倍
- 需要稳定的网络连接
- CPU 版本功能不受影响，升级失败可继续使用

### 8.5 金融知识学习中心

主页点击"金融学习"按钮，可展开学习中心：

- **入门基础**：股票/债券/基金基础概念、官方投教平台链接
- **进阶分析**：技术指标详解、基本面分析框架
- **高级量化**：Python 金融编程、量化交易课程资源
- **常用术语速查**：PE/PB/ROE/MACD/RSI 等核心概念的大白话解释

## 九、插件系统

### 9.1 Kronos K 线预测

- 基于 Kronos-base 深度学习模型（102.3M 参数，MIT 开源）
- 使用 Monte Carlo 采样进行概率性价格区间预测
- 预测结果注入 AI 多智能体辩论，并交叉验证
- 在"配置概览"中可开关
- 无 GPU 时自动使用 CPU 模式，无显卡也能运行

### 9.2 FinBERT 情绪分析

- 基于 FinBERT 金融文本情绪分析模型
- 对新闻和公告进行批量情绪评分
- 模型不可用时自动降级为规则引擎
- 在"配置概览"中可开关

## 十、常见问题

### 软件相关

**Q: 软件打不开，命令行窗口一闪而过**
A: 查看 `%USERPROFILE%\.quantsage\logs\quantsage.log` 日志文件。常见原因：端口被占用、权限不足。尝试重启电脑后重试。

**Q: 浏览器没有自动打开**
A: 查看命令行窗口显示的 URL（如 `http://127.0.0.1:8501`），手动在浏览器输入。

**Q: 显示"Streamlit 无法连接"**
A: 等待 10-20 秒后刷新浏览器页面。如果持续无法连接，重启软件。

**Q: 软件占用空间很大**
A: 安装包约 1.4 GB，主要包含 AI 模型权重（~406 MB）和 Python 运行环境。如需释放空间，卸载后手动删除 `%USERPROFILE%\.quantsage\`。

### 分析相关

**Q: 分析到一半卡住不动了**
A: 分析需要 3-5 分钟。如果超过 10 分钟无响应，可能是 DeepSeek API 繁忙，重启软件后重试。

**Q: 分析结果不完整，只有部分章节**
A: 尝试将分析深度调至 3 或更低。如果问题持续，联系开发者。

**Q: 没有 Kronos 预测章节**
A: 可能原因：(1) Kronos 插件未开启（检查配置概览）；(2) K 线数据不足（需至少 30 个交易日数据）；(3) 模型权重文件缺失（重新安装）。

**Q: 基本面 PE/PB 显示"暂不可用"**
A: 免费数据接口（BaoStock/AKShare）不提供实时 PE/PB。如需此数据，需配置 Tushare Token（在配置向导中填写）。

### 激活相关

**Q: 设备码在哪里看**
A: 软件主页顶部或激活页面均有显示。
